# 🚀 Modern Data Platform for E-Commerce Analytics

## 📌 Overview
This project builds a real-world data engineering pipeline using Azure, Databricks, and Delta Lake.

## 🧱 Architecture
Raw Data → Bronze → Silver → Gold → Power BI

## ⚙️ Tech Stack
- Azure Data Lake
- Databricks (PySpark)
- Delta Lake
- Azure Data Factory
- Power BI

## 🔥 Features
- Streaming pipeline (real-time simulation)
- Incremental data processing
- Delta Lake merge operations
- Data modeling (Fact & Dimension tables)

## 📊 Dashboard
- Revenue trends
- Top products
- Customer insights
